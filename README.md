# Netflix Exploratory Data Analysis (EDA)

## Overview
This project performs an exploratory data analysis (EDA) on the Netflix titles dataset to understand content distribution, trends, and patterns across movies and TV shows.

The analysis focuses on:
- Content type distribution (Movies vs TV Shows)
- Year-wise release trends
- Country-wise content contribution
- Rating-wise distribution
- Initial genre exploration
- Handling missing values and data cleaning

This is an **iterative project**, with insights and depth increasing across days.

---

## Dataset
- Source: Netflix Titles Dataset
- Rows: 8,807
- Columns: 12
- Each row represents a unique movie or TV show available on Netflix.

## Day 1 Highlights
- Data inspection and structure analysis
- Missing value handling

- Year-wise content trends
- Country-wise analysis (filtered for major contributors)
- Rating-wise analysis with Movie vs TV Show split
- Initial genre extraction logic

---

## Key Insights (So Far)
- Movies dominate the Netflix catalog, but TV shows show strong growth in later years.
- The US and India are the largest content contributors.
- Mature ratings (TV-MA, TV-14) dominate both movies and TV shows.
- Clear regional and rating-based differences exist in content focus.
